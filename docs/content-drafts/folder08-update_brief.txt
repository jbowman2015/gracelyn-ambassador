WHAT THIS FILE IS FOR
=============================================================

Agent 3 reads this file for the Week 4 "Program Update" email in the standard
four-week content calendar (docs/design/Gracelyn_Agent_3_Engagement_v2.md
Section 5.2). Claude personalizes this content per ambassador but the
substance, program news and forward-looking message, comes from here.

This is NOT a one-time deliverable. It is meant to be refreshed periodically
by Parmeet so the Week 4 email always reflects real, current program news
instead of going stale. If this file is ever missing or empty, Agent 3's
design doc says Claude generates a generic mission message instead, so
nothing breaks, but ambassadors get a less specific, less interesting email
until this is updated.

Recommended refresh cadence: monthly, or whenever there is real news
(a program milestone, a new track, a season closing out).

Keep entries following ambassador_copy_rules.txt: no "commission," no em
dashes, mission before any numbers, no admissions-counseling language.
